## Project Overview

Image Captioning is a Deep Learning application that automatically generates meaningful textual descriptions for input images. This project combines Computer Vision and Natural Language Processing techniques to understand image content and produce human-readable captions.

A Convolutional Neural Network (CNN) extracts visual features from the image, while a Long Short-Term Memory (LSTM) network generates captions word by word based on those extracted features.

---

## Objectives

- Automatically generate captions for images.
- Extract image features using a pretrained VGG16 CNN model.
- Train an LSTM model to generate natural language descriptions.
- Improve image understanding using Deep Learning.

Dataset Structure:

dataset/

---

## Working Process

1. Load image dataset.
2. Extract image features using VGG16.
3. Store extracted features.
4. Process captions.
5. Create tokenizer.
6. Train LSTM caption generator.
7. Save trained model.
8. Predict captions for new images.
