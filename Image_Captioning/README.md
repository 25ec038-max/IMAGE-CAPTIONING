IMAGE CAPTIONING USING DEEP LEARNING
Project Description
Title
Image Captioning using CNN and LSTM

Abstract
Image Captioning is a Deep Learning application that automatically generates a textual description for an image. It combines Computer Vision and Natural Language Processing (NLP) techniques. A Convolutional Neural Network (CNN) extracts image features, while a Long Short-Term Memory (LSTM) network generates meaningful captions word by word.

This project uses the VGG16 pre-trained CNN model for feature extraction and an LSTM network for sentence generation.

Features
Upload any image

Extract image features using VGG16

Generate captions automatically

Uses Deep Learning

Supports Flickr8K Dataset

Easy to train and test
